# AmazeMC

Official website for the AmazeMC Minecraft server — a Survival Economy realm running Minecraft 1.20.4.

## Features

- Server status with live player count (via mcsrvstat.us API)
- Staff directory, server rules, voting links
- Donor ranks with tiered perks
- In-game store with keys, currency, and bundles
- Server guide with commands, economy info, and land claiming
- Items & tools showcase with Minecraft rarity system
- Minecraft-themed UI with custom cursor, loading screen, and particle effects

## Dev

Run a local server with:

```bash
npm run dev
```

Or just open `index.html` directly in any browser.
